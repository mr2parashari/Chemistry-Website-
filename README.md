# 🧪 Chemistry Learning Web App

A simple interactive Chemistry learning website built using **HTML, CSS, and JavaScript**.  
This project helps students learn Chemistry in a structured and visual way with topics and quizzes.

---

## 🚀 Live Demo
👉 https://chem-project-teal.vercel.app

---

## 📚 Features

### 🧬 1. Organic Chemistry
- 5 structured topics
- 10 interactive quizzes
- Easy concept explanation
- Visual learning support

---

### ⚗️ 2. Inorganic Chemistry
- 5 structured topics
- 10 quizzes
- Periodic table-based learning
- Reaction concepts

---

### 🔬 3. Physical Chemistry
- 5 main topics
- 10 quizzes
- Formula-based learning
- Numerical practice

---

## 🧩 UI Structure

- 📊 Grid-based dashboard
- 📁 Separate sections for each branch:
  - Organic Chemistry
  - Inorganic Chemistry
  - Physical Chemistry

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

---

## 🎯 Purpose

This project is designed for:
- School / College students
- Quick revision of Chemistry concepts
- Practice quizzes for better understanding
